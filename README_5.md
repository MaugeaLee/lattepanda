# 5. Booting Disk install
- 현재 라떼판다의 부팅은 오롯이 USB 플래시 디스크에서 수행하고 있다.
- USB를 제거한다면 부팅이 되지 않는다
- 따라서 USB 부팅 디스크의 부팅 수행 능력을 SSD or HDD로 옮기고 이를 수행하도록 설정해주어야 한다.
